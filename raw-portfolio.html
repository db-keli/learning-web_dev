My Projects

  • Fingerprint Attendance System

    Contributed to a team's final year project where we built a Fingerprint Attendance System that uses the RFID and a fingerprint sensor to verify student attendance. Using Streamlit I contributed to preparing the data and building of the web application. I also contributed to the electronics of the system using the ESP32.
